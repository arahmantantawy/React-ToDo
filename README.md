# Simple To Do List (React)

This is a beginner React project that displays a simple to-do list.  
It was created as a practice project while learning the basics of React.

## Features

- Simple page layout
- Title at the top of the page
- Add items to a to-do list
- Built using React and JavaScript

## Purpose

The goal of this project is to practice:
- React components
- JSX
- Basic state management
- Project structure in a React app

## Technologies Used

- React
- Vite
- JavaScript
- HTML
- CSS

## Status

This project is a learning exercise and may be expanded in the future.
